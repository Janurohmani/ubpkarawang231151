Generate 1000 Data Siswa dengan Mockaroo

Langkah-langkah untuk Menghasilkan dan Mengimpor Data

1. Kunjungi Mockaroo

Buka situs web Mockaroo.

2. Atur Skema Data

Tambahkan field sesuai dengan struktur tabel siswa Anda:

  • id: (Type: Row Number)
  • nama: (Type: Full Name)
  • umur: (Type: Number, rentang 10-18)
  • kelas: (Type: Custom List, misalnya '10A', '10B', '11A', '11B')
  • alamat: (Type: Street Address)

3. Generate Data

Atur jumlah baris data yang ingin Anda hasilkan menjadi 1000 dan klik tombol Download Data untuk mengunduh file CSV.

4. Persiapkan Database

Pastikan Anda memiliki tabel siswa di database Anda. Berikut adalah contoh struktur tabel SQL:

CREATE TABLE siswa (
    id INT PRIMARY KEY AUTO_INCREMENT,
    nama VARCHAR(100),
    umur INT,
    kelas VARCHAR(10),
    alamat VARCHAR(255)
);

5. Import Data ke Database

Gunakan perintah berikut untuk mengimpor data dari file CSV ke tabel siswa:

LOAD DATA INFILE '/path/to/your/file.csv'
INTO TABLE siswa
FIELDS TERMINATED BY ','
ENCLOSED BY '"'
LINES TERMINATED BY '\n'
IGNORE 1 ROWS;

Gantilah /path/to/your/file.csv dengan jalur yang sesuai ke file CSV yang telah Anda unduh.

Contoh Query INSERT Manual

Jika Anda ingin melakukan INSERT secara manual, berikut adalah contoh format query:

INSERT INTO siswa (nama, umur, kelas, alamat) VALUES
('John Doe', 15, '10A', '123 Main St'),
('Jane Smith', 16, '10B', '456 Elm St'),
('Alice Johnson', 14, '11A', '789 Oak St');

Kesimpulan

Dengan menggunakan Mockaroo, Anda dapat dengan mudah menghasilkan 1000 data siswa yang realistis untuk pengujian dan pengembangan. Setelah menghasilkan data, Anda dapat mengimpor data tersebut ke dalam tabel siswa di database sekolah Anda menggunakan perintah SQL yang sesuai.
